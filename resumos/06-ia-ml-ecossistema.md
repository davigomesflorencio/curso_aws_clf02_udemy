# Tópico 06: IA/ML, Ecossistema e Serviços Especializados

## 1. Inteligência Artificial (AI) e Machine Learning (ML)
A AWS oferece uma pilha completa de serviços de IA/ML, desde infraestrutura para especialistas até serviços prontos para uso por desenvolvedores.

*   **Amazon SageMaker:** Plataforma totalmente gerenciada para construir, treinar e implantar modelos de ML rapidamente.
*   **Amazon Bedrock:** Serviço gerenciado para construir e escalar aplicações de IA Generativa com FMs (Modelos de Fundação).
*   **Serviços de IA Prontos (Ready-to-use):**
    *   **Amazon Rekognition:** Análise de imagem e vídeo (reconhecimento facial, objetos).
    *   **Amazon Polly:** Converte texto em fala realista.
    *   **Amazon Lex:** Tecnologia para construir chatbots (mesma tecnologia da Alexa).
    *   **Amazon Transcribe:** Converte fala em texto.
    *   **Amazon Translate:** Tradução fluente de idiomas.
    *   **Amazon Comprehend:** Processamento de Linguagem Natural (NLP) para encontrar insights em textos.

## 2. Serviços de Integração e Especializados
*   **AWS Step Functions:** Orquestrador de workflows serverless que permite conectar diversos serviços AWS em etapas lógicas (State Machines).
*   **AWS Cognito:** Gerenciamento de identidade para aplicações web e móveis. Controle de login, cadastro e acesso de usuários (User Pools e Identity Pools).
*   **AWS IoT Core:** Permite conectar dispositivos de IoT à nuvem AWS de forma fácil e segura, processando e roteando trilhões de mensagens.
*   **Amazon WorkSpaces:** Serviço de Desktop como Serviço (DaaS). Fornece áreas de trabalho virtuais (Windows ou Linux) baseadas na nuvem para os usuários.

## 3. Ecossistema AWS
*   **AWS Marketplace:** Um catálogo digital com milhares de listagens de software de fornecedores independentes (ISVs) que rodam na AWS. Facilita encontrar, testar, comprar e implantar software.
*   **AWS Partner Network (APN):** Comunidade global de parceiros que utilizam a AWS para construir soluções e serviços para clientes.
    *   **Consulting Partners:** Ajudam clientes a projetar, arquitetar, migrar e gerenciar cargas de trabalho.
    *   **Technology Partners:** Fornecem soluções de software que são hospedadas ou integradas à AWS.
*   **AWS Knowledge Center:** Repositório de perguntas frequentes e ajuda técnica oficial.

## 4. Programas de Migração e Suporte à Inovação
*   **AWS CAF (Cloud Adoption Framework):** Organiza o processo de migração para a nuvem em 6 perspectivas (Business, People, Governance, Platform, Security, Operations).
*   **AWS IQ:** Permite encontrar e contratar especialistas certificados pela AWS para projetos sob demanda.
